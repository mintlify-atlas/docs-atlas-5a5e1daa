# Standard Theme Variant

The **Standard** variant is the original Shades of Purple theme that has been used by over 5 million developers worldwide. It features a carefully balanced color palette with hand-picked shades of purple designed to reduce eye strain during long coding sessions.

<Info>
This is the default theme variant and is optimized for most lighting conditions and use cases.
</Info>

## What Makes it Special

The Standard variant offers:

- **Perfect Balance**: Mid-tone purple backgrounds that work well in various lighting conditions
- **High Contrast**: Carefully selected foreground colors for excellent readability
- **Professional Look**: Polished appearance ideal for presentations and screencasts
- **Eye Comfort**: Color palette optimized to reduce eye strain during extended coding
- **Universal Appeal**: Works great for all programming languages and file types

## Activation Instructions

<Note>
Make sure you have installed the Shades of Purple theme extension from the VSCode marketplace.
</Note>

### Method 1: Using the Command Palette

1. Open the Command Palette with `Cmd/Ctrl + Shift + P`
2. Type "Color Theme" and select **Preferences: Color Theme**
3. Choose **Shades of Purple** from the list

### Method 2: Via Settings

Add this to your `settings.json`:

<CodeGroup>
```json settings.json
{
  "workbench.colorTheme": "Shades of Purple"
}
```
</CodeGroup>

### Method 3: Quick Open

1. Press `Cmd/Ctrl + K` then `Cmd/Ctrl + T`
2. Select **Shades of Purple** from the theme list

<Info>
The theme takes effect immediately without requiring a restart.
</Info>
